ADM Bike Woo Locations

Plugin para WooCommerce que administra cobertura de envíos mediante una jerarquía de Estado → Municipio → Código Postal.

Objetivo

Permitir reglas de envío más precisas para tiendas que necesitan controlar disponibilidad, costo o bloqueo por ubicación.

Incluye

  • Gestión de estados, municipios y códigos postales.
  • Reglas de envío por cobertura, costo o bloqueo.
  • Validación en carrito y checkout.
  • Integración con WooCommerce.

Estructura

  • README.md: guía principal del plugin.
  • docs/index.html: página simple de documentación.
Esta carpeta puede crecer después con capturas, guías de instalación y ejemplos de configuración.
