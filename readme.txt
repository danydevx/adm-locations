Advanced WooCommerce shipping coverage management by State, Municipality and Postal Code.

== Description ==

ADM Bike Woo Locations lets you define shipping coverage using a hierarchy of State -> Municipality -> Postal Code.

Features include:

* Guided checkout selectors for state, municipality and postal code
* Shipping rules for free, paid and unavailable coverage
* Exact postal code, postal range, municipality and state matching
* Conflict detection for overlapping rules

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/admbike-woo-locations/`.
2. Activate the plugin from the Plugins screen.
3. Configure states, municipalities and postal codes from ADM Bike Locations.
4. Add shipping rules and enable the shipping method in WooCommerce.

== FAQ ==

= Does this plugin require WooCommerce? =

Yes. WooCommerce must be active.

= Can I use it without custom coverage data? =

The plugin works best when you define states, municipalities and postal codes.
